# GRA Implementierung
Die GRA (Grid-Reference Architecture) von Prosume basiert auf das Agentensystem JADE. Die einzelnen Akteure sind als Agents definiert, die über Nachrichten miteinander kommunzieren. Ziel ist es einen vollständigen Satz des Datenaustausches zu erhalten und die Handelsfunktionen für einen P2P Strommarkt abzubilden.
Der Einsatz ist auf die Forschung unter Laborbedingungen beschränkt, wobei die Veränderungen in der Kommunikation zu verschiedenen Marktszenarien simuliert werden können

# Limitierungen
Die in diesem Projekt umgesetzte Implementierung dient nicht zur Verwendung in einem realen Szenario, da besonders die Absicherungsfunktionen für die Kommunikation bislang nicht implementiert wurden.
